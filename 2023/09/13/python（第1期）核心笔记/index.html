（1）fabs（）与abs（）函数的区别

在python中，fabs（x）也是返回x的绝对值，虽然两个函数功能类似，但是存在以下差异：

◎abs（）是一个内置函数，而fabs（）是在math模块中定义的。

◎fabs（）函数只适用于浮点数和整数类型，而abs（）函数参数可以是实数（整数、浮点数等），还可以是复数（如果是复数，返回复数的模）。

（example）用法：

>>>print(abs(-100))

100

\>>>import math **#****调用****math****模块****
**>>>print(math.fabs(-100)) **#****在math中调用fabs（x）函数**

100.0

（2）divmod（）函数

◎divmod（x，y），如果x，y都是整数，那么采用整数除法，运算结果相当于（x//y，x%y）。如果x或y是浮点数，则运算结果相当于（math.floor(x/y,x%y）。

（example）：

>>>print(divmod(2,6))

(0,2)

>>>print(divmod(28,7))

(4,7)

>>>print(divmod(25.6,2))

(12.0, 1.6000000000000014)

>>>print(divmod(2,0.2))

(9.0, 0.1999999999999999)

（3）pow(x,y[,z])函数

◎pow（x,y[,z]）函数相当于x的y次方，如果z存在，在对结果进行取模，等效于pow（x，y）%z。其中pow（x，y）与x**y等效。一起计算提升计算效率，但要求3个参数必须是数值类型。

（example）：

>>>print(pow(2,3))

8

>>>print(pow(2,3,5))

3

（3）字符串的查找

str.find(sub[,start[,end]])函数是基础的字符串查找函数。如果在字符串中未查找到指定字符，会返回-1；否则返回指定字符第一次出现位置的序列，第一个字符序列为0.

（example）：

>>>str="Microsoft Corporation"
>>>print(str.find('p'))

13

◎也可以指定查找开始和结束的位置，格式为：str.find(需要查找的字符，开始位置，结束位置)

(example):

>>>str="Microsoft Corporation"
>>>print(str.find('o',5)) **#****表示从本字符串的第****5****个字符****‘s’****开始向右查找****'o'**

6

 
>>>str="Microsoft Corporation"
>>>print(str.find('o',5,7)) **#****表示从本字符串的第****5****个字符****‘s’****开始向右查找****'o'****直到第****7****个字符****'f'****结束**

7

◎还可以使用str.rfind(sub[,start[,end]])函数从右往左查找。此时，返回值仍为第一次查找到的字符序列。其他用法均与str.find()类似。

（example）：

>>>str="Microsoft Corporation"
>>>print(str.rfind('t'))

17

 

◎count（）函数用于统计字符串中指定字符出现的次数。

（example）：

>>>str="Microsoft Corporation"
>>>print(str.count('t'))

2

第三章 控制程序执行流程

程序运行的基本结构分为3种，分别是顺序结构、分支结构、循环结构。

*1.顺序结构*

顺序结构指的是在程序执行过程中按照所有语句的顺序逐一执行的程序。

（实例1）：求圆的周长和面积

>>>r=eval(input("please insert the R"))
>>>s=3.14*r*r
>>>l=2*3.14*r
>>>print("圆的面积为：","%.2f" %s)
>>>print("圆的周长为：","%.2f" %l)

a=eval(input("请输入正方形的边长："))

2. *分支结构*

在python中，可以通过if、elif、else等条件判断语句来实现单分支、双分支和多分支结构。

（1）单分支结构（注意：python中没有switch-case语句）

单分支结构由3部分组成：关键字if、用于判断结构真假的条件表达式，以及调节表达式为真时执行的语句块。

语法：

if<条件表达式>:

<语句块>

注意的问题：

①在每个条件后面要使用冒号（:），表示接下来是满足条件后要执行的语句块。

②使用缩进来划分语句块，多条具有相同缩进数的语句组成一个语句块。

③在单分支结构中，也可以并列使用多条if语句实现对不同条件的判断。

（实例1）：判断年龄

age=eval(input("请输入你的年龄："))
if age>=18:
  print("您的年龄是：",age)
  print("成年人")
if age<18:
  print("您的年龄是：",age)
  print("未成年人")

请输入你的年龄：16

您的年龄是： 16

未成年人

（实例2）：判断PM2.5

PM=eval(input("请输入PM2.5的值："))
if 0<=PM<50:
  print("空气质量优")
if 50<=PM<100:
  print("空气质量良")
if PM>=100:
  print("空气质量差")

请输入PM2.5的值：180

空气质量差

（3）多分支结构

在python中，用if-elif-else描述多分支结构。

语法：

if<条件表达式1>:

语句块1

elif<条件表达式2>:

语句块2

elif<条件表达式3>:

语句块3

…

else:

<语句块N+1>

注意点：

①无论有多少个分支，程序在执行了一个分支后，其余分支不再执行。

②elif不能写成elseif。

（实例）：成绩等级评定

score=eval(input("请输入你的驾驶证理论考试成绩："))
if score>=90:
  print("您的成绩评级为：A级")
elif 90>score>=80:
  print("您的成绩评级为：B级")
elif 80>score>=70:
  print("您的成绩评级为：C级")
elif 70>score>=60:
  print("您的成绩评级为：D级")
else:
  print("您的成绩评级为：E级")

请输入你的驾驶证理论考试成绩：68

您的成绩评级为：D级

（1）遍历循环（有限循环）：for语句

for语句通常由两部分组成，分别是条件控制部分和循环部分。

（2）无限循环（条件循环）：while语句

无限循环一直保持循环操作直到特定的循环条件不被满足才结束。无需知道循环次数。

语法：

<语句块1>

Python中要实现循环的自由转场，就需要用到break和continue辅助控制循环。

      break
